# Lin-UI 定义文件

Lin-UI API 的 TypeScript 类型定义文件

## 安装

需要将 lin 文件夹复制到微信小程序的 TypeScript 类型定义文件夹

在微信小程序的 TypeScript 类型定义文件（`index.d.ts`）中加入一行语句（`/// <reference path="./lin/index.d.ts" />`）即可完成安装
